# Khalid21456-Chatbot-Microsoft-DialoGPT-Medium

This project implements a simple chatbot using Flask and the Microsoft DialoGPT-medium model. It allows users to send messages and receive responses from the chatbot through a web interface. The chatbot utilizes the `transformers` library to interact with the DialoGPT-medium model for generating conversational responses.

## Features

-   Web-based chat interface using Flask
-   Integration with Microsoft DialoGPT-medium model for generating responses
-   Display of user and bot messages in a chat format
-   Automatic scrolling to the bottom of the chat window

## Requirements

-   Python 3.6 or higher
-   Flask
-   Transformers
-   PyTorch
